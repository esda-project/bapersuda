⚙️ Konfigurasi Supabase

Hubungkan SIANGGARAN ke database Supabase Anda untuk penyimpanan data terpusat dan multi-pengguna.

📋 Langkah setup:
1. Buat project di supabase.com
2. Copy Project URL dan anon public key dari Settings → API
3. Jalankan SQL migration di bawah ini di SQL Editor Supabase
4. Klik Simpan & Sambungkan
SQL Migration — jalankan di Supabase SQL Editor
