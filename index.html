1. Buat project di supabase.com
2. Copy Project URL dan anon public key dari Settings → API
3. Jalankan SQL migration di bawah ini di SQL Editor Supabase
4. Klik Simpan & Sambungkan

🎵 Cara Mendapat URL Lagu

          
URL audio harus berakhiran .mp3, .ogg, atau .wav dan bisa diakses publik.

        


        

          
📦 Sumber Lagu Gratis Bebas Hak Cipta

          

            

              
1. Pixabay Music ✅ Gratis & bebas royalti

              
Buka pixabay.com/music → cari genre → klik lagu → klik kanan tombol Download → "Copy link address" → paste ke kolom URL.

            

            

              
2. Free Music Archive ✅ Gratis CC License

              
Buka freemusicarchive.org → pilih lagu → klik kanan tombol Download MP3 → "Copy link address".

            

            

              
3. GitHub / Google Drive / OneDrive

              
Upload file MP3 ke GitHub (raw URL) atau Google Drive (ubah link ke format direct download). Pastikan file bisa diakses tanpa login.

            

            

              
4. Supabase Storage ✅ Rekomendasi untuk file pribadi

              
Upload MP3 ke Supabase Storage bucket (set public) → copy URL publik → paste di sini. Cocok untuk lagu Jawa / Indonesia koleksi sendiri.

            

          

        


        

          
⚠️ YouTube & Spotify

          
URL dari YouTube / Spotify tidak bisa digunakan langsung karena bukan file audio langsung. Gunakan sumber yang memberikan link file .mp3 langsung.

        


        

          
💡 Contoh URL yang Valid

          
            https://cdn.pixabay.com/audio/2022/05/27/audio_1808fbf07a.mp3

            https://files.freemusicarchive.org/storage.../track.mp3

            https://raw.githubusercontent.com/user/repo/main/lagu.mp3
